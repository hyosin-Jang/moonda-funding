# moonda-funding
## 클래스 한 줄 기대평 웹사이트
- 이 사이트는 오프라인 클래스를 개설하기 전에 클래스 한 줄 기대평을 받아 각 클래스에 대한 사용자의 수요를 확인할 수 있는 웹사이트입니다. 
- 메인페이지, 클래스 상세 페이지로 구성되어 있으며 응원하기 버튼을 클릭하고 모달창에 핸드폰 번호와 기대 한 마디를 남기면 추후에 클래스가 개설 될 때 알림 메시지를 받을 수 있습니다.
